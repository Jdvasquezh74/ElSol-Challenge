Comandos de prueba:

Upload ->

Previo, cd C:/Ruta/Carpeta hacia la carpeta donde se encuentre el archivo de audio...

curl -X POST "http://localhost:8000/api/v1/upload-audio" -H "accept: application/json" -H "Content-Type: multipart/form-data" -F "file=@AudioPrueba.mp3"

Transcriptions Database (ORM) ->

curl -X GET "http://localhost:8000/api/v1/transcriptions"
curl -X GET "http://localhost:8000/api/v1/transcriptions/{transcription_id}"
curl -X GET "http://localhost:8000/api/v1/transcriptions/5a08f898-b7ee-4387-878a-4b3251192ce6"


Vector Database ->

curl -X GET "http://localhost:8000/api/v1/vector-store/status"
curl -X GET "http://localhost:8000/api/v1/vector-store/conversations?limit=10"
curl -X GET "http://localhost:8000/api/v1/vector-store/conversations/f201ded2-712d-4b91-8a19-254a251236d5"

Chat RAG ->

curl -X POST "http://localhost:8000/api/v1/chat" -H "accept: application/json" -H "Content-Type: application/json" -d "{\"query\": \"¿Qué enfermedad tiene Juan Pérez?\", \"max_results\": 5, \"include_sources\": true}"

Documento PDF ->

curl -X POST "http://localhost:8000/api/v1/upload-document" -H "accept: application/json" -H "Content-Type: multipart/form-data" -F "file=@documento_juan_carlos_perez.pdf" -F "patient_name=Juan Carlos Pérez" -F "document_type=Receta médica"
curl -X GET "http://localhost:8000/api/v1/documents"

Speaker Diarization ->
curl -X POST "http://localhost:8000/api/v1/upload-audio" -H "accept: application/json" -H "Content-Type: multipart/form-data" -F "file=@AudioPrueba.mp3"
